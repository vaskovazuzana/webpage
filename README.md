# Ing. Zuzana Vašková — Personal Portfolio Website

## Project Overview

A personal professional portfolio website for **Ing. Zuzana Vašková** — a building design engineer and architect-oriented professional focused on reconstruction projects, project documentation, and BIM modeling.

This repository contains the planning, concept, and content structure for a multilingual static website to be hosted on **Cloudflare Pages**.

## Website Goals

1. Present Zuzana as a credible professional in building design and BIM
2. Serve as a high-quality digital business card
3. Open space for future collaboration and networking opportunities
4. Show what she works on and how she thinks
5. Include a light personal layer without becoming a lifestyle blog
6. Build long-term personal brand value

## Typography Direction

- Clean, minimalist, sans-serif-first
- Clear, modern, restrained, elegant, readable
- Hierarchy through spacing, scale, weight, and rhythm
- No ornamental, luxury-fashion, or cold-tech-startup typography
